# Connect Four: Deep Space Edition 🚀

A modern, visually striking take on the classic Connect Four game. Built entirely with vanilla web technologies, this project features a deep-space UI, glowing neon game pieces, and smooth drop animations.

## ✨ Features
* **Sleek Neon UI:** A dark `#07091a` space-themed background with pulsing, glowing discs.
* **Smooth Physics:** Custom JavaScript animation handles the realistic dropping of discs into the board.
* **Game Logic:** Full win/draw detection checking horizontally, vertically, and diagonally.
* **Responsive Design:** Optimized for both desktop and mobile screens with touch-support enabled.
* **Zero Dependencies:** No frameworks or libraries—just pure HTML, CSS, and ES6+ JavaScript.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom variables, grid layout, and keyframe animations.
* **JavaScript:** Object-Oriented design, DOM manipulation, and request-based physics logic.
